# Documentation
A simple News app.

## Pages
 - Home = /
 - Add News page = /news
 - News = /news:id
 - Edit News= /news/edit/:id
 - Not Found Page = "*"

### Functionality
Created a `useCommentOnchange()` hook that manages state for controlled form for the comment
Created a `useNewsOnchange()` hook that manages state for controlled form for the News
